# reactNativeCourseAssignment1

## Animated Counter Application

A React Native mobile application that demonstrates interactive counter functionality with beautiful, fluid animations. Built using Expo for cross-platform development.

## What This App Does

This counter app allows users to increase or decrease a numeric value through intuitive button interactions. The display number animates with smooth scaling and opacity transitions whenever the count changes, creating an engaging user experience.

### Setup Instructions

Clone or download this repository, then:

1. Open terminal and navigate to the project folder:
```bash
cd reactNativeCourseAssignment1
```

2. Install all required packages:
```bash
npm install
```

### Launching the Application

To start the development server, run:
```bash
npx expo start
```
